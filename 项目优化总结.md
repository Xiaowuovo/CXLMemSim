### 4. 自动化实验脚本

#### 脚本1: `scripts/run_experiments.sh`

**功能**：
- 自动运行所有标准实验
- 生成CSV结果文件
- 生成实验对比报告
- 支持快速模式和详细模式

**用法**：
```bash
# 运行所有实验
./scripts/run_experiments.sh

# 快速模式（测试用）
./scripts/run_experiments.sh --quick

# 指定输出目录
./scripts/run_experiments.sh --output-dir ./my_results
```

**输出文件**：
```
results/
├── experiment_results_20240321_143000.csv
├── experiment_results_20240321_143000.json
└── experiment_report_20240321_143000.txt
```

#### 脚本2: `scripts/plot_results.py`

**功能**：
- 读取实验CSV数据
- 生成论文级别的图表
- 支持PDF和PNG双格式输出
- 自动生成LaTeX/Markdown表格

**生成的图表**：
1. **图5-1**: 延迟敏感性分析（柱状图）
2. **图5-2**: 带宽影响分析（折线图）
3. **图5-3**: 拥塞模型对比（堆叠柱状图）
4. **图5-4**: MLP优化效果对比（柱状图）
5. **图5-5**: CXL访问比例（饼图）
6. **表格**: 实验结果汇总表（LaTeX+Markdown）

**用法**：
```bash
# 从CSV生成所有图表
python3 scripts/plot_results.py results/experiment_results.csv

# 指定输出目录
python3 scripts/plot_results.py results/data.csv --output-dir ./figures
```

## 📊 预期实验结果

基于CXL技术特性和学术文献，预期实验结果：

### 延迟敏感性
- 本地DDR: ~90ns
- CXL Gen5: ~170ns (+89%)
- CXL Gen4: ~250ns (+178%)
- 远距离: ~350ns (+289%)

**论文讨论点**：延迟与性能呈近似线性关系

### 带宽影响
- 64GB/s: 基线延迟
- 32GB/s: +10-15%
- 16GB/s: +20-30%
- 8GB/s: +40-50%

**论文讨论点**：带宽瓶颈下拥塞模型效果显著

### 拥塞模型效果
- 禁用: 基础延迟
- 启用: +15-25%（低带宽场景）

**论文讨论点**：拥塞模型更接近真实硬件行为

### MLP优化
- 禁用: 基线
- 启用: -10-15%

**论文讨论点**：并行优化可掩盖部分延迟
