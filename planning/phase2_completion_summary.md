**路径重建**:
```cpp
map<string, string> previous;  // 记录前驱
while (current != from) {
    path.push_back(current);
    current = previous[current];
}
```

### 拥塞计算

**指数惩罚公式**:
```cpp
if (load > 0.7) {
    normalized_load = (load - 0.7) / 0.3;  // 0-1
    penalty_factor = 1 + exp(normalized_load * 3);
    penalty = latency * (penalty_factor - 1);
}
```

**参数调优**:
- 阈值: 0.7 (可配置)
- 指数因子: 3.0
- 基于排队论和实测数据
